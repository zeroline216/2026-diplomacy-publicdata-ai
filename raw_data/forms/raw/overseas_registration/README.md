# overseas_registration 원본 서식 폴더

이 폴더는 재외공관 재외국민등록 민원 처리에 필요한 원본 서식 파일을 저장하는 공간이다.

재외국민등록 민원은 국가별로 schema를 분리하지 않고, 공통 schema와 공관별 policy를 통해 필요한 서식을 선택한다.

이 폴더에는 재외국민등록신청서, 변경·이동 신고서, 등록부등본 교부신청서, 위임장 등 재외국민등록 신청과 증명서 발급에 필요한 공통·공관별 서식이 함께 포함된다.

## 폴더 경로

```text
raw_data/forms/raw/overseas_registration/
```
