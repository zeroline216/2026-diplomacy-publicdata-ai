# family_relation 원본 서식 폴더

이 폴더는 재외공관 가족관계 민원 처리에 필요한 원본 서식 파일을 저장하는 공간이다.

가족관계 민원은 국가별로 schema를 분리하지 않고, 공통 schema와 공관별 policy를 통해 필요한 서식을 선택한다.

이 폴더에는 가족관계등록사항별 증명서 교부 신청서, 위임장, 영문증명서 발급 신청서, 우편수령 신청서 등 가족관계 증명서 발급과 관련된 공통·공관별 서식이 함께 포함된다.

## 폴더 경로

```text
raw_data/forms/raw/family_relation/
```
