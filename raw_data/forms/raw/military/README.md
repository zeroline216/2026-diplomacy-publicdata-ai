# military 원본 서식 폴더

이 폴더는 재외공관 병역 민원 처리에 필요한 원본 서식 파일을 저장하는 공간이다.

병역 민원은 국가별로 schema를 분리하지 않고, 공통 schema와 공관별 policy를 통해 필요한 서식을 선택한다.

이 폴더에는 국외여행허가 신청서, 기간연장 허가 신청서, 위임장, 재외국민2세 확인신청서, 재학사실 확인서, 동일인확인서, 개인정보 제공 동의서 등 병역의무자의 국외체류·연장허가·사실확인 관련 서식이 포함된다.

## 폴더 경로

```text
raw_data/forms/raw/military/
```
