# 2026-diplomacy-publicdata-ai (2026 외교 공공데이터 AI 활용경진대회)

## 프로젝트명
재외공관 민원 서류 준비 및 자동작성 도우미

## 프로젝트 개요
해외 체류 국민이 여권, 가족관계, 공증, 병역, 비자, 긴급 상황 관련 민원을 처리할 때 필요한 서류와 절차를 AI가 맞춤형으로 안내하는 서비스입니다.

## 주요 기능
- 사용자 상황 입력
- 필요한 민원 서류 안내
- 재외공관별 절차 안내
- 서류 작성 초안 생성
- 관련 공공데이터/API 활용

## 팀원 역할

## 👥 팀원 역할

### 한여진 · 임아진 (데이터 파이프라인)
- 외교부 공공데이터 API 수집
- 영사민원24 및 재외공관 페이지 스크래핑
- RAG 학습용 문서(코퍼스) 구축 및 관리

### 진영선 · 연소윤 (AI 및 서비스 개발)
- 민원유형 분류 및 RAG 기반 질의응답 구현
- 신청서 초안 생성 및 LLM 검증 로직 개발
- 웹 서비스 구현 및 API 연동
- 시스템 평가, 발표자료 및 보고서 작성

## 사용 기술
- Python
- FastAPI
- React
- OpenAI API
- 외교부 공공데이터
