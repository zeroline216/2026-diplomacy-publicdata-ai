# 2026-diplomacy-publicdata-ai (2026 외교 공공데이터 AI 활용경진대회)

## 프로젝트명
재외공관 민원 서류 준비 및 자동작성 도우미

## 프로젝트 개요
해외 체류 국민이 여권, 가족관계, 공증, 병역, 비자, 긴급 상황 관련 민원을 처리할 때 필요한 서류와 절차를 AI가 맞춤형으로 안내하는 서비스입니다.

## 주요 기능
- 사용자 상황 입력
- 필요한 민원 서류 안내
- 재외공관별 절차 안내
- 서류 작성 초안 생성
- 관련 공공데이터/API 활용

## 팀원 역할

| 팀원 | 담당 분야 | 주요 역할 |
|------|----------|----------|
| 한여진 | 데이터 파이프라인 | 외교부 공공데이터 API 연동, 영사민원24 및 재외공관 페이지 데이터 수집·스크래핑, RAG 학습용 코퍼스 구축 및 관리 |
| 임아진 | AI 백엔드 | 민원유형 분류 로직 구현, RAG 검색 및 답변 생성 파이프라인 구축, 신청서 초안 생성 및 LLM 검증 로직 개발 |
| 진영선 | 프론트엔드 | 사용자 입력 화면 및 결과 페이지 구현, 백엔드 API 연동, 웹 서비스 UI/UX 개발 |
| 연소윤 | 평가 및 프로젝트 관리 | 평가 데이터셋 구축, 성능 평가 및 결과 분석, 발표자료·보고서 작성, 프로젝트 일정 및 문서 관리 |

## 사용 기술
- Python
- FastAPI
- React
- OpenAI API
- 외교부 공공데이터
