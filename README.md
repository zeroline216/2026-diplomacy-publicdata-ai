# 2026-diplomacy-publicdata-ai

# 2026 외교 공공데이터 AI 활용경진대회

## 프로젝트명
재외공관 민원 서류 준비 및 자동작성 도우미

## 프로젝트 개요
해외 체류 국민이 여권, 가족관계, 공증, 병역, 비자, 긴급 상황 관련 민원을 처리할 때 필요한 서류와 절차를 AI가 맞춤형으로 안내하는 서비스입니다.

## 주요 기능
- 사용자 상황 입력
- 필요한 민원 서류 안내
- 재외공관별 절차 안내
- 서류 작성 초안 생성
- 관련 공공데이터/API 활용

## 팀원 역할
| 이름 | 역할 |
|---|---|
| 팀원1 | 기획 |
| 팀원2 | 데이터/API 조사 |
| 팀원3 | 프론트엔드 |
| 팀원4 | 백엔드/AI |

## 사용 기술
- Python
- FastAPI
- React
- OpenAI API
- 외교부 공공데이터
